1、团队成员个人信息

该页面主要为 成员个人主页，默认统一设计，如果某成员有自主设计的个人主页，可以直接提供。

准备材料：

  • 证件照（生活照）一张

  • 个人简介

    • 英文
    • 介绍自身研究方向、教育经历

  • 个人联系方式等（自愿）

    • email
    • google scholar
    • github
    • blog

  • publication

    • 提供内容
      • 论文标题
      • 作者列表（自己加粗）
      • 会议/期刊全称 + （缩写）+ 年份
      • [arxiv]
      • [github]
      • [Published version]
        • 文章正式发表的链接
      • [Supplementary materials]
        • 文章如果有附加材料的链接
      • [slide]
        • 文章如果做了报告，ppt的链接
      • [talk]
        • 文章如果录制了视频，链接
  • 文字类信息梳理好放置于 word或markdown文档，以个人名字命名
  • 照片单独，以个人名字命名
